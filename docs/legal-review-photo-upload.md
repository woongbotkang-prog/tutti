# 사진 업로드 법적 리스크 검토

## 현재 상태
- 공고당 최대 3장, 장당 5MB (JPG/PNG/WebP)
- Supabase Storage `gig-images` 버킷 (public read)

## 리스크 항목

### 1. 초상권 침해
- 타인 사진 무단 업로드 시 초상권 침해 (민법 제750조 불법행위)
- 플랫폼은 중개자 위치 → 직접 책임은 낮으나, 통지 후 미조치 시 방조 가능

### 2. 저작권 침해
- 공연/콘서트 사진 (타 사진가 촬영물) 무단 사용
- 악보 사진 업로드 → 출판사 저작권 침해

### 3. 개인정보보호법 (PIPA)
- 사진에 얼굴 포함 시 개인정보에 해당
- 개인정보처리방침에 '이미지 수집·이용' 항목 명시 필요

## 권장 조치 (우선순위)

### 즉시 (MVP)
- [ ] 업로드 시 안내 문구: "업로드하는 사진의 저작권과 초상권 책임은 게시자에게 있습니다"
- [ ] 이용약관에 면책 조항 추가

### 단기
- [ ] 신고/삭제 기능 (부적절한 이미지 신고 버튼)
- [ ] 개인정보처리방침에 이미지 관련 내용 추가

### 중기
- [ ] EXIF 메타데이터 자동 제거 (GPS 정보 등)
- [ ] 이미지 모더레이션 (부적절한 이미지 자동 탐지)
