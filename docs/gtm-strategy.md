## 1. 핵심 과제

TUTTI는 클래식 연주자 매칭 플랫폼이다. 개인 연주자와 단체(오케스트라, 실내악)를 연결하며, **곡 기반 프로젝트 모집**이라는 차별화 기능을 갖고 있다.

### 닭-달걀 문제

모든 양면 플랫폼의 숙명:

- 공고가 없으면 → 연주자가 안 온다
- 연주자가 없으면 → 단체가 공고를 안 올린다

### 해결 전략: 공급 먼저 쌓기

**"배달의민족 초기 전략"** — 직접 발로 뛰어서 아마추어 오케스트라 공고를 대신 올려준다.

1. 단체 공고를 인위적으로 먼저 확보 (대리 등록)
2. 공고가 쌓인 상태에서 개인 연주자 유입
3. 연주자가 모이면 단체가 자발적으로 공고를 올리기 시작

## 3. 단계별 실행 계획

### Phase 0: 론칭 전 (D-30 ~ D-1)

**목표: 공고 최소 20개 확보 (단체 10개 + 개인 10개)**

#### Week 1-2: 단체 리스트업 & 접촉

- [ ] 네이버/페이스북/인스타에서 아마추어 오케스트라 **50개 리스트업**
  - 단체명, 활동 지역, 연락처(DM/이메일), 최근 활동 여부
- [ ] DM/이메일로 접촉 시작: "무료로 단원 모집 공고 올려드릴게요"
- [ ] 응답 없는 단체 → 2차 접촉 (다른 채널 or 직접 방문)
- **목표: 10개 단체 공고 확보**

#### Week 3-4: 개인 연주자 공략

- [ ] 네이버 카페 3곳 이상에 서비스 소개 게시글 작성
- [ ] TUTTI 인스타그램 계정 개설 & 운영 시작
- [ ] 클래식 연주자 100명 팔로우 & 관심 표현
- [ ] 음대 커뮤니티 접촉 (에브리타임, 학교 카페)
- **목표: 개인 연주자 50명 사전 등록**

### Phase 1: 론칭 (D-Day ~ D+30)

- [ ] 소셜미디어 공식 론칭 발표 (인스타, 페이스북)
- [ ] 사전 등록자에게 론칭 알림 발송
- [ ] 초기 사용자 "개척자 배지" 부여 (프로필에 표시)
- [ ] 첫 주 매일 1개 이상 새 공고 등록 유지 (직접 등록 포함)
- [ ] 사용자 피드백 수집 (카카오톡 오픈채팅 or 구글폼)

### Phase 2: 성장 (D+30 ~ D+90)

- [ ] 매칭 성공 사례 콘텐츠화 (인스타 카드뉴스, 스토리)
- [ ] 사용자 후기 수집 & SNS 공유
- [ ] 입소문 전략: "친구 초대 시 프로필 하이라이트" 등
- [ ] 곡 기반 프로젝트 모집 기능 적극 홍보 (차별화 포인트)
- [ ] 단체 자발적 공고 등록 유도 (대리 등록 → 직접 등록 전환)

## 4. 직접 영업 스크립트

### DM 템플릿 (단체 접촉용)

#### 버전 A: 네이버 카페/인스타 DM

```
안녕하세요! TUTTI라는 클래식 연주자 매칭 플랫폼을 운영하고 있어요.

[단체명] 오케스트라의 단원 모집 공고를 저희 플랫폼에 무료로 올려드리고 싶어요.
네이버 카페나 SNS보다 더 많은 연주자분들께 노출될 수 있도록 도와드릴게요.

공고 내용만 주시면 제가 직접 올려드릴게요 😊
관심 있으시면 답장 부탁드립니다!
```

#### 버전 B: 현재 모집 중인 단체용 (더 구체적)

```
안녕하세요! [단체명]에서 [파트명] 단원을 모집하고 계신 걸 봤어요.

저희 TUTTI에서 무료로 공고를 올려드리면, 클래식 연주자분들에게 더 넓게 노출돼요.
카페 공고 내용 그대로 옮겨드릴게요 — 추가 작업 전혀 없습니다!

올려드려도 괜찮을까요?
```

#### 버전 C: 후속 메시지 (무응답 시 3일 후)

```
안녕하세요, 지난번 메시지 보내드렸던 TUTTI입니다!
혹시 단원 모집에 도움이 필요하시면 편하게 말씀해주세요.
무료이고, 공고 내용만 주시면 제가 다 해드려요 😊
```

### 접촉 시 핵심 포인트

1. **무료** — 비용 부담 없음을 강조
2. **대신 해드림** — 단체 입장에서 추가 노력 제로
3. **더 많은 노출** — 기존 채널 대비 이점

## 5. 콘텐츠 전략 (SNS)

### 핵심 채널: 인스타그램

클래식 연주자들이 가장 활발하게 활동하는 SNS.

| 포맷 | 콘텐츠 | 빈도 |
|------|--------|------|
| **피드** | 매칭 성공 스토리, 연주 현장 사진 (동의 후) | 주 2-3회 |
| **릴스** | "클래식 연주자라면 알아야 할 것", "단원 모집 팁" | 주 1회 |
| **스토리** | 새 공고 알림, 투표/퀴즈 | 매일 |
| **카루셀** | "서울 아마추어 오케스트라 TOP 10" 등 정보성 | 주 1회 |

### 콘텐츠 아이디어 (론칭 전후)

- "서울에서 아마추어 오케스트라 찾는 법"
- "단원 모집 공고, 이렇게 쓰면 지원율 2배"
- "클래식 연주자 사이드 프로젝트 모음"
- "이 곡 같이 연주할 사람?" (곡 기반 프로젝트 홍보)

### 보조 채널

- **네이버 카페 게시글**: 서비스 소개 + 유용한 정보 제공
- **카카오톡 오픈채팅**: 초기 사용자 소통 & 피드백

## 6. KPI (핵심 성과 지표)

### 론칭 30일 후 목표

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 등록 공고 | 50개 | DB 집계 |
| 가입 사용자 | 200명 | 회원가입 수 |
| 지원 발생 | 30건 | 지원 액션 수 |
| 매칭 성공 (수락) | 10건 | 양측 수락 수 |

### 3개월 후 목표

| 지표 | 목표 |
|------|------|
| 등록 공고 | 200개 |
| 가입 사용자 | 1,000명 |
| MAU (월간 활성 사용자) | 300명 |
| 단체 자발적 공고 비율 | 30% 이상 |

### 핵심 퍼널

```
방문 → 가입 → 프로필 완성 → 공고 열람 → 지원 → 매칭 성공
```

각 단계 전환율을 추적하고, 병목 구간을 개선한다.

## 7. 예산 계획 (저비용 전략)

### MVP 단계: 거의 0원

| 항목 | 비용 |
|------|------|
| Vercel (호스팅) | 무료 |
| Supabase (DB/Auth) | 무료 (500MB, 50K 요청/월) |
| 도메인 | ~15,000원/년 |
| 인스타 광고 (선택) | ~10만원/월 이하 |
| **월 합계** | **0 ~ 10만원** |

### 유료 전환 시점

| 트리거 | 전환 | 비용 |
|--------|------|------|
| MAU 300명 초과 | Supabase Pro | $25/월 |
| 상업적 운영 | Vercel Pro | $20/월 |
| 사용자 1,000명+ | 이메일/푸시 알림 서비스 | ~$10/월 |

**원칙:** 사용자가 먼저, 비용은 나중에. PMF(Product-Market Fit) 확인 전까지 최소 비용 유지.

## 9. 즉시 실행 가능한 액션 리스트

### 오늘 할 수 있는 것 ✅

1. **네이버 카페** "아마추어 오케스트라 모임" 검색 → 상위 10개 단체 리스트업
2. **인스타 계정 개설** (@tutti.music 또는 @tutti_kr)
3. **첫 DM 발송** 3개 (시범 — 버전 A 템플릿 사용)
4. **구글 스프레드시트** 생성: 단체 리스트 관리용 (단체명 / 채널 / 연락처 / 상태)
5. **경쟁사 조사**: 유사 서비스 있는지 30분 리서치

### 이번 주 안에 할 것 📋

6. 리스트업 50개 단체 완성
7. DM 20개 이상 발송
8. 인스타 첫 게시물 3개 업로드
9. 론칭 랜딩 페이지 최종 점검
10. 사전 등록 폼 준비 (구글폼 or 자체)
