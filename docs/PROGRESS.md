## 확정된 핵심 결정사항 요약

| 항목 | 결정 |
|------|------|
| 플랫폼명 | TUTTI |
| 타겟 | 한국 우선, 향후 일본 |
| 온보딩 | 가입→프로필(나중에 가능)→탐색→공고 |
| 비회원 | 탐색 O, 지원 X |
| 프로필 없으면 | 지원 X |
| 실력 레벨 | 5단계 (입문~전문가) |
| 거절 사유 | 객관식 + 주관식(옵션) 필수 |
| 지원 인원 제한 | 없음 |
| 악기 | 오케스트라 전 악기 |
| 지역 | 시/도 단위, 온라인 없음 |
| 레퍼토리 | 장르태그 + 작곡가 자동완성 + 곡명 자유입력 |
| 공고 만료 | 직접 설정 + 최대 90일 자동만료 |
| 매너온도 | 채팅 후 N일(기본30), 블라인드 |
| 알림 | 이메일 + 웹푸시 (카카오 Phase 2) |
| 소셜 로그인 | 이메일 + 카카오 + 구글 |
| 채팅 | 텍스트만 (MVP) |
| 단체 관리자 | 1명 |
| 수익모델 | 초기 무료 → 부스팅/프리미엄 구독 |
| 기술스택 | Next.js + Supabase + Vercel |
