> 작성일: 2026-02-13 | PM: 웅 + AI

## 온보딩 플로우 (확정)
```
회원가입 → 프로필 작성 (나중에 하기 가능) → 공고 탐색 → 공고 올리기 (옵셔널)
```

### 결정 사항
- [x] 프로필 작성은 필수가 아님 (나중에 하기 가능)
- [x] 가입 직후 공고 탐색 가능 (read-only)
- [x] 공고 올리기는 옵션 (프로필 작성 후에만 가능하게 할지 검토 필요)

### 미결 사항
- [x] 프로필 없이 지원하기 가능한가? → ❌ 프로필 필수 (확정)
- [x] 비회원 공고 열람? → ✅ 가능 (확정, SEO 유리)
- [ ] 나중에 하기 클릭 시 → 홈(공고 피드)으로 이동?
- [ ] 공고 올리기 전 프로필 필수 여부?

## 사용자 스토리 (작성 중)

### US-001: 회원가입
*As a* 클래식 연주자
*I want to* 이메일 또는 카카오로 가입하고 싶다
*So that* TUTTI 서비스를 이용할 수 있다

**Acceptance Criteria:**
- 이메일 + 비밀번호 가입 가능
- 카카오 소셜 로그인 가능
- 가입 시 개인/단체 유형 선택
- 가입 완료 후 프로필 작성 화면으로 이동 (나중에 하기 버튼 있음)

---

### US-002: 프로필 작성 (개인)
*As a* 개인 연주자
*I want to* 내 악기, 실력, 레퍼토리를 입력하고 싶다
*So that* 나를 찾는 단체/개인이 나를 파악할 수 있다

**실력 레벨 (확정):**
- 🟢 입문: 레슨 1~2년 미만
- 🔵 초급: 레슨 2~4년, 간단한 앙상블 가능
- 🟡 중급: 레슨 4년+, 아마추어 오케스트라 가능
- 🟠 고급: 음대 재학/졸업 수준
- 🔴 전문가: 현역 프로 연주자

**Acceptance Criteria:**
- 필수: 닉네임, 주 악기, 실력 레벨, 지역
- 선택: 사진, 경력, 레퍼토리, 연습 빈도, 영상 링크
- 저장 후 내 프로필 페이지 확인 가능
- 언제든 수정 가능

---

### US-003: 프로필 작성 (단체)
*As a* 단체 운영자
*I want to* 우리 단체를 소개하고 싶다
*So that* 우리 단체에 맞는 연주자를 찾을 수 있다

**Acceptance Criteria:**
- 필수: 단체명, 단체 유형, 지역, 현재 편성
- 선택: 로고, 설명, 연습 일정, 영상/사진

---

### US-004: 공고 탐색
*As a* 사용자 (프로필 없어도 가능)
*I want to* 공고 목록을 탐색하고 싶다
*So that* 나에게 맞는 앙상블/연주자를 찾을 수 있다

**Acceptance Criteria:**
- 로그인 없이도 공고 목록 열람 가능 (비회원)
- 필터: 악기, 단체형태, 지역, 실력레벨, 장르
- 공고 카드: 제목, 단체명/개인, 악기, 지역, 등록일 표시
- 공고 상세 클릭 → 전체 내용 + 지원하기 버튼

---

### US-005: 공고 등록
*As a* 프로필을 작성한 사용자
*I want to* 구인/구직 공고를 올리고 싶다
*So that* 매칭 파트너를 찾을 수 있다

**지원 플로우 (확정):**
```
지원하기 + 한줄 메시지
→ 등록자에게 알림
→ 등록자가 프로필 확인 후 수락/거절
→ 수락: 채팅방 자동 개설
→ 거절: 거절 사유 선택 필수 (객관식 + 주관식 옵션)
```

**거절 사유 객관식 (초안):**
- 실력 수준이 맞지 않아요
- 지역/연습 장소가 맞지 않아요
- 연습 빈도/강도가 맞지 않아요
- 레퍼토리 방향이 달라요
- 이미 멤버를 찾았어요
- 기타 (주관식 입력)

**지원 인원 제한:** 없음 (확정)

**Acceptance Criteria:**
- 프로필 작성 완료 후에만 공고 등록 가능
- 구인 (단체→개인 찾기) / 구직 (개인→팀 찾기) 선택
- 공고 내용: 제목, 찾는 악기, 실력, 레퍼토리, 지역, 연습빈도, 마감일
- 등록 후 즉시 피드에 노출

## 지역 단위 (확정)
- 시/도 단위 (서울, 경기, 인천, 부산, 대구, 광주, 대전, 울산, 세종, 강원, 충북, 충남, 전북, 전남, 경북, 경남, 제주)
- 온라인 옵션 없음

## 레퍼토리 입력 방식 (확정)
3단계 구조:
1. **장르/시대 태그**: 바로크 / 고전 / 낭만 / 근현대 / 현대
2. **작곡가 검색**: 자동완성 (~200명 주요 작곡가 사전 DB)
3. **곡명**: 자유 입력 (옵션)

→ 작곡가 기반 정밀 매칭 가능, 나중에 곡 DB 확장 예정

## 공고 유효기간 (확정)
- 등록자가 직접 마감일 설정 + 최대 90일 자동 만료

## 매너온도 평가 (확정)
- 채팅방 개설 후 **N일 뒤** 평가 요청 (기본값 30일, 추후 조정 가능한 변수로 관리)
- 블라인드 방식: 내가 먼저 평가 완료해야 상대 점수 공개

- 알림 트리거: 새 지원자 / 수락·거절 / 채팅 메시지 / 평가 요청
