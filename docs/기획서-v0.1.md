# TUTTI 기획서 v0.1
> 클래식 연주자 매칭 플랫폼
> 작성일: 2026-02-13 | 상태: 초안

---

## 1. 서비스 개요

### 한 줄 정의
> "클래식 연주자와 앙상블·오케스트라를 연결하는 매칭 플랫폼"

### 배경 & 문제
- 아마추어/프로 클래식 연주자들은 협연 파트너를 구하기 위해 SNS, 커뮤니티 게시판, 지인 네트워크에 의존
- 체계적인 필터링 없이 악기·실력·레퍼토리·지역이 맞는 파트너를 찾기 매우 어려움
- 단체(오케스트라, 실내악 그룹) 역시 결원 발생 시 마땅한 구인 채널 없음

### 솔루션
악기·실력·레퍼토리·지역·연습강도 기반의 정밀 매칭 + 신뢰도 시스템

---

## 2. 타겟 사용자

### 메인 타겟
- **개인 연주자**: 현악기(바이올린·비올라·첼로·더블베이스), 관악기, 피아노 등 클래식 연주자
  - 아마추어 (취미): 레슨 이수자, 음악 동호인
  - 세미프로: 음대 재학·졸업생, 프리랜서 연주자
  - 프로: 오케스트라 단원, 실내악 연주자

- **단체/조직**:
  - 아마추어 오케스트라
  - 실내악 그룹 (스트링 콰르텟, 피아노 트리오 등)
  - 청소년 오케스트라
  - 음악 학원·단체

### 초기 지역 타겟
- 한국 (서울·수도권 우선 → 전국 확대)
- 향후: 일본 (고전음악 인구 많음, 문화적 유사성)

## 3. 핵심 기능 명세

### 3.1 회원가입 & 프로필

#### 개인 연주자 프로필
- 기본 정보: 이름/닉네임, 지역, 사진
- 악기 정보: 주 악기, 보조 악기
- 실력 레벨: 입문 / 초급 / 중급 / 고급 / 전문가
- 음악 경력: 학력(음대 여부), 레슨 이력, 주요 활동
- 레퍼토리: 연주 가능 곡 목록 (자유 입력 + 태그)
- 선호 앙상블 형태: 오케스트라 / 실내악 / 둘 다
- 연습 빈도: 주 1회 / 주 2-3회 / 주 4회 이상
- 연습 강도: 캐주얼 / 보통 / 진지함
- 자기소개 (자유 텍스트)
- 연주 영상 링크 (YouTube, Instagram 등)

#### 단체/조직 프로필
- 단체명, 로고, 설명
- 단체 유형: 오케스트라 / 실내악 그룹 / 합창단 / 기타
- 현재 구성원 수 및 악기 편성
- 지역 (연습 장소)
- 지향하는 음악 스타일
- 연습 일정·빈도·강도
- 대표 연주 영상/사진

---

### 3.2 공고 시스템

#### 단체 → 구인 공고 (멤버 모집)
```
제목: [콰르텟] 비올라 구합니다
단체: 강남 하모니 스트링 콰르텟
찾는 악기: 비올라
요구 실력: 고급 이상
예정 레퍼토리: 슈베르트 D.956, 베토벤 Op.18
연습: 매주 일요일 오후, 강남구
지원 조건: 음대 재학 이상 선호
마감일: 2026-03-15
```

#### 개인 → 구직 공고 (팀 찾기)
```
제목: [바이올린] 실내악 그룹 찾습니다
악기: 바이올린
실력: 중급-고급
원하는 형태: 스트링 트리오 또는 콰르텟
관심 레퍼토리: 낭만파, 현대음악
지역: 마포구, 온라인 가능
연습 빈도: 주 1-2회
```

#### 공고 필터 (검색)
- 악기 종류
- 단체 형태 (오케스트라/실내악/등)
- 실력 레벨
- 지역
- 연습 빈도/강도
- 레퍼토리 장르 (바로크/고전/낭만/현대)

---

### 3.3 매칭 & 지원 시스템

**플로우:**
1. 공고 발견 → 프로필 확인
2. 지원하기 버튼 클릭 → 간단한 지원 메시지 작성
3. 공고 등록자에게 알림
4. 수락 → 채팅창 자동 개설
5. 거절 → 정중한 거절 알림

---

### 3.4 채팅 시스템
- 매칭 성사 후 1:1 채팅 (개인↔단체)
- 단체 내부 그룹 채팅 (멤버 전용)
- 파일 공유 (악보 PDF 등)
- 연습 일정 공유

---

### 3.5 매너 온도 (신뢰도 시스템)
- 초기 온도: 36.5°C
- 협연 완료 후 상호 평가
- 평가 항목:
  - 약속 이행 (연습 참석률)
  - 실력이 프로필과 일치하는가
  - 태도·매너
  - 재협연 의향
- 온도 범위: 0°C ~ 100°C
- 높은 온도 = 신뢰할 수 있는 연주자/단체
- 특정 온도 이하: 가입 제한 등 패널티 검토

---

### 3.6 탐색 & 커뮤니티 (Phase 2)
- 연주자 목록 브라우징
- 악보 공유 게시판
- 연주 영상 피드
- 공연 정보 게시판

## 4. 사용자 플로우

### 개인 연주자
```
회원가입 → 프로필 작성 → 공고 탐색 or 내 공고 등록
→ 마음에 드는 단체/개인 발견 → 지원
→ 수락 → 채팅 → 협연 → 상호 평가
```

### 단체/조직
```
회원가입 → 단체 프로필 작성 → 구인 공고 등록
→ 지원자 검토 → 수락/거절
→ 채팅 → 협연 → 상호 평가
```

## 6. DB 설계 (초안)

### 주요 테이블

**users** (공통)
- id, email, password_hash, type(individual/organization), created_at

**individual_profiles**
- user_id, name, region, instruments[], level, bio, video_url
- preferred_ensemble_type, practice_frequency, practice_intensity
- manner_temp (default: 36.5)

**organization_profiles**
- user_id, org_name, org_type, region, description, logo_url
- member_count, practice_day, practice_frequency, practice_intensity
- manner_temp

**listings** (공고)
- id, author_id, author_type, title, description
- listing_type (recruiting/seeking), instrument_wanted
- level_required, repertoire_tags[], genre_tags[]
- region, practice_frequency, deadline, status(active/closed)
- created_at

**applications** (지원)
- id, listing_id, applicant_id, message, status(pending/accepted/rejected)
- created_at

**chats**
- id, listing_id, participant_ids[], created_at

**messages**
- id, chat_id, sender_id, content, type(text/file), created_at

**reviews** (평가)
- id, reviewer_id, reviewee_id, collaboration_id
- score_reliability, score_skill_match, score_manner
- would_collaborate_again (bool), comment, created_at

## 7. 기술 스택

| 영역 | 기술 | 이유 |
|------|------|------|
| Frontend | Next.js 14 (App Router) | SSR/SEO, 모바일 친화 |
| Styling | Tailwind CSS + shadcn/ui | 빠른 UI 개발 |
| Backend | Next.js API Routes | 풀스택 단일 레포 |
| Database | Supabase (PostgreSQL) | 인증+DB+실시간+스토리지 통합 |
| 실시간 채팅 | Supabase Realtime | 별도 서버 불필요 |
| 인증 | Supabase Auth | 소셜로그인 포함 |
| 파일 저장 | Supabase Storage | 악보, 프로필 이미지 |
| 배포 | Vercel | 자동 배포, 무료 티어 |
| 도메인 | tutti.kr (or tutti.co.kr) | 확인 필요 |

## 8. 수익 모델 로드맵

### Phase 1 (0~6개월): 무료
- 전면 무료, 사용자 확보 집중
- KPI: MAU 500명, 공고 200건, 매칭 50건

### Phase 2 (6~12개월): 부분 유료화
1. **공고 부스팅** — 공고 상단 노출 (1회 2,000~5,000원)
2. **단체 프리미엄** — 월 19,900원
   - 무제한 공고 등록
   - 단체 인증 배지
   - 지원자 상세 필터
3. **연주자 인증 배지** — 연 9,900원
   - 음대 졸업, 콩쿨 수상 등 증빙 후 인증

### Phase 3 (12개월+): 확장
4. **공연 티켓팅 연동** — 티켓 판매 수수료 5%
5. **악보 마켓플레이스** — 연주자들이 편곡 악보 판매
6. **일본 진출** — tutti.jp
