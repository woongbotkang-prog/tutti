# TUTTI 플랫폼 - 전체 화면 와이어프레임

클래식 연주자 ↔ 앙상블/오케스트라 매칭 서비스  
**기술 스택**: Next.js + Tailwind CSS + shadcn/ui  
**설계 원칙**: 모바일 우선, 직관적 네비게이션, 명확한 CTA

---

- **법적 동의**: 최소 필수 약관만 포함
- **빠른 회원가입 경험**: 단계 선택 페이지 없이 한 화면에서 해결

- **질문 기능**: 지원 전 궁금한 점 물어볼 수 있음
- **명확한 CTA**: 지원하기 버튼이 가장 아래 배치 (스크롤 후 결정)
- **프로필 없는 사용자 유도**: "프로필 완성하기" 링크 제공

## 19. 모바일 하단 내비게이션 바

```
┌─────────────────────────────────┐
│                                 │
│  [콘텐츠 영역]                  │
│                                 │
├─────────────────────────────────┤
│  🏠 홈   🔍 탐색  💬 채팅      │
│  👤 마이   🔔 알림              │ (총 5개 탭)
│                                 │
│  활성: 파란색(#2563EB)          │
│  비활성: 회색(#9CA3AF)          │
│  라벨: 텍스트 표시              │
│                                 │
└─────────────────────────────────┘

각 아이콘 설명:
- 🏠 홈: 맞춤 공고 피드 (메인)
- 🔍 탐색: 공고 검색/필터 (구인/구직)
- 💬 채팅: 진행 중인 채팅/채팅 요청
- 👤 마이: 마이페이지 (프로필 보기/관리)
- 🔔 알림: 모든 알림 (탭별 필터링)

배치 전략:
┌──────────────────────────────┐
│ [홈] [탐색]  [채팅]  [마이]   │  (4개)
│     [알림 배지]              │  (중앙 상단)
└──────────────────────────────┘

또는 (권장)

┌──────────────────────────────┐
│ [홈] [탐색] [채팅] [마이] [알림]│ (5개)
│  활성  활성  활성  비활성 배지  │
└──────────────────────────────┘
```

### UX 설계 의도
- **5개 탭**: 핵심 기능 모두 포함 (피드, 탐색, 채팅, 마이페이지, 알림)
- **명확한 라벨**: 아이콘 아래 텍스트 라벨로 혼동 방지
- **배지 표시**: 읽지 않은 채팅/알림 개수를 배지로 표시
- **항상 접근 가능**: 어떤 화면에서든 하단 네비로 이동 가능
- **터치 친화성**: 손가락으로 누르기 쉬운 크기 (최소 44x44px)
